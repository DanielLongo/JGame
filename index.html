What is it?

It's a library for the Java language that allows beginning developers to easily develop 2D games.

Who is it for?

It's for anyone who has a bit of experience in Java programming and is willing to learn along the way.

How does it work?

You create a game using the JGame framework, and, behind-the-scenes, JGame integrates with AWT to run your game. You never need to worry about a line of AWT code.
